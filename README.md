# Kelompok 1 Internship Finder - Mobile App

Aplikasi mobile untuk mencari dan melamar lowongan magang - Tugas Besar DPPB

## Features

### Authentication (Hidayat)
- Splash Screen dengan auto navigation
- Login dengan validasi
- Register akun baru

### Home & Search (Arsyad)
- Bottom Navigation (4 tabs)
- List lowongan magang
- Search & Filter
- Detail lowongan

### Profile & Application (Dzikra)
- Form lamaran lengkap
- Riwayat lamaran dengan status
- Profile & Edit Profile
- Logout
