## Cara Mengambil Screenshot

### Windows
1. Tekan `Win + Shift + S`
2. Pilih area yang mau di-screenshot
3. Paste ke Paint, save dengan nama file yang sesuai

### Flutter DevTools
1. Run aplikasi: `flutter run`
2. Buka DevTools, klik icon camera

### Android Emulator
1. Klik icon camera di toolbar emulator
2. Atau gunakan `Ctrl + S`

## Format File

- **Format:** PNG (recommended)
- **Size:** Max 2MB per file
- **Naming:** Lowercase dengan underscore (contoh: `01_splash.png`)
